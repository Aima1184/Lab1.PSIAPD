## 1. Сверточная модель

В данной лабораторной работе мы сосредоточимся на реализации сверточной нейронной сети AlexNet, которая стала одним из первых успешных примеров глубокого обучения для обработки изображений. AlexNet была представлена в 2012 году и продемонстрировала значительное улучшение точности по сравнению с предыдущими методами в конкурсе ImageNet.

### 1.1 Архитектура AlexNet

![image](https://neurohive.io/wp-content/uploads/2018/10/AlexNet-1-1.png)

Рисунок 1 – Архитектура сети AlexNet

Архитектура AlexNet состоит из восьми слоев с весовыми коэффициентами: 

1. **Сверточный слой 1**: Применяет 96 фильтров размером 11x11 с шагом 4, что позволяет извлекать основные признаки из изображения.
2. **Слой подвыборки (Max Pooling)**: Уменьшает пространственные размеры выходных данных, что помогает уменьшить количество параметров и вычислительную сложность.
3. **Сверточный слой 2**: Использует 256 фильтров размером 5x5, что позволяет извлекать более сложные признаки.
4. **Слой подвыборки (Max Pooling)**: Снова применяется для уменьшения размерности.
5. **Сверточный слой 3**: Применяет 384 фильтра размером 3x3, что позволяет выявлять еще более детализированные признаки.
6. **Сверточный слой 4**: Использует 384 фильтра размером 3x3.
7. **Сверточный слой 5**: Применяет 256 фильтров размером 3x3.
8. **Полносвязный слой 1**: Содержит 4096 нейронов, который обрабатывает извлеченные признаки.
9. **Полносвязный слой 2**: Также содержит 4096 нейронов.
10. **Выходной слой**: Содержит 1000 нейронов (для классификации на 1000 классов).

Каждый сверточный слой использует функцию активации ReLU, что позволяет избежать проблемы затухающего градиента и ускоряет обучение.

### 1.2 Оптимизатор AdaSmoothDelta

Оптимизатор AdaSmoothDelta представляет собой адаптивный алгоритм, который корректирует скорость обучения для каждого параметра на основе истории градиентов и изменений параметров. Основные шаги его работы:

1. **Инициализация**: Для каждого параметра модели инициализируются скользящие средние для градиентов и изменений параметров.
2. **Обновление градиентов**: На каждой итерации вычисляются градиенты функции потерь по отношению к параметрам модели.
3. **Смягчение изменений**: Используются скользящие средние для градиентов и изменений, что позволяет уменьшить резкие колебания и обеспечивает более плавное и устойчивое схождение.
4. **Обновление параметров**: Параметры обновляются с учетом адаптивной скорости обучения, что позволяет оптимизатору более эффективно находить минимум функции потерь.

### Математические аспекты оптимизации

Для расчета обновлений параметров в AdaSmoothDelta используются следующие формулы:

- Обновление градиента:
  $$
  g_t = \beta_1 g_{t-1} + (1 - \beta_1) \nabla L(\theta_t)
  $$
- Обновление изменений парамет
- Обновление изменений параметров:
  \[
  v_t = \beta_2 v_{t-1} + (1 - \beta_2) \nabla L(\theta_t)^2
  \]
- Обновление параметров:
  \[
  \theta_{t+1} = \theta_t - \frac{\alpha}{\sqrt{v_t} + \epsilon} g_t
  \]

где \( \alpha \) — скорость обучения, \( \epsilon \) — малое значение для предотвращения деления на ноль, \( \beta_1 \) и \( \beta_2 \) — коэффициенты для скользящих средних.
